# Bannerlord.HeroManagerEnhanced

Hero-focused QoL combo mod integrating:

- **Auto Ammo Pickup** — automatic battlefield ammo collection
- **Perk Concierge (GimeAllPerks)** — grant all perks to the player
- **Adjustable Leveling** — customizable XP curves, skill rates, and smithing research

## Dependencies

- Bannerlord.Harmony
- Bannerlord.ButterLib
- Bannerlord.UIExtenderEx
- Bannerlord.MBOptionScreen (MCM)

## Build

```powershell
./dev/build.cmd -Version v1.0.0
```

Output: `out/Bannerlord.HeroManagerEnhanced/` and zip package.

## MCM

Three separate settings entries under Mod Options:

- Auto Ammo Pickup
- Perk Concierge
- Adjustable Leveling

## Localization

English (EN) and Simplified Chinese (CNs) included.
